# QUALITY_SCORE.md

## 목적

이 문서는 카드, 직업, 몬스터, 보스 기믹, 합동기, 필살기 기획을 평가하기 위한 기준을 정의한다.

핵심 목표는 재미있는 아이디어를 유지하면서도 한 달 마스터 범위 안에서 구현 가능하고, 수치적으로 과도하지 않은 설계를 만드는 것이다.

## 카드 밸류 기본 점수

카드 효과는 아래 기준으로 1차 점수화한다.

| 효과 | 기준 점수 |
| --- | --- |
| 피해 1 | 1점 |
| 방어도 1 | 1점 |
| 방어력 증가 1, 1턴 | 1점 |
| 카드 드로우 1장 | 1.5점 |
| 디버프 1턴 | 1점 |

계산 예시:

```text
피해 5 + 방어도 3 = 8점
방어력 증가 2, 1턴 + 피해 4 = 6점
피해 4 + 카드 1장 드로우 = 5.5점
디버프 2턴 + 방어도 5 = 7점
```

방어도는 턴 종료 시 사라지는 임시 보호막이고, 방어력 증가는 버프로 관리되는 피해 감소다. 방어력 증가가 2턴 이상 지속되면 지속 턴 수만큼 밸류를 계산한다.

## 카드 등급별 최대 밸류

카드는 1~3코스트 범위에서 설계한다.

### Normal / Status Card

노말 카드와 상태이상 카드는 안정적이고 예측 가능한 효과를 기준으로 한다.

| 코스트 | 최대 밸류 |
| --- | --- |
| 1 Cost | 6점 |
| 2 Cost | 14점 |
| 3 Cost | 22점 |

### Rare Card

희귀 카드는 조건부 시너지, 높은 효율, 빌드 방향성을 제공한다.

| 코스트 | 최대 밸류 |
| --- | --- |
| 1 Cost | 7점 |
| 2 Cost | 20점 |
| 3 Cost | 25점 |

### Legendary Card

전설 카드는 런의 전략을 바꿀 수 있지만, 최초 마스터에서는 수량을 제한한다.

| 코스트 | 최대 밸류 |
| --- | --- |
| 1 Cost | 9점 |
| 2 Cost | 22점 |
| 3 Cost | 30점 |

## 밸류 초과 허용 조건

카드가 기준 밸류를 초과하려면 아래 조건 중 하나 이상이 필요하다.

- 특정 직업 전용 카드다.
- 특정 상태 스택이 필요하다.
- 대상이 제한된다.
- 이번 턴 이후 패널티가 있다.
- 필드 조건이 필요하다.
- 행동 큐에서 특정 순서를 요구한다.
- 합동기 또는 필살기 조건 일부로 설계된다.

단, 조건이 있더라도 플레이어가 이해하기 어려우면 좋은 설계로 보지 않는다.

## 카드 평가 체크리스트

카드 기획은 아래 항목을 확인한다.

| 항목 | 질문 |
| --- | --- |
| 밸류 | 코스트/등급 대비 점수가 기준 안에 있는가? |
| 역할 | 공격, 방어, 드로우, 디버프 중 역할이 명확한가? |
| 직업성 | 검사/마법사/힐러 중 해당 직업의 정체성을 강화하는가? |
| 선택지 | 항상 정답인 카드가 아닌가? |
| 큐 활용 | 행동 큐 순서에 따라 가치가 달라질 여지가 있는가? |
| UI | 카드 텍스트가 짧고 이해 가능한가? |
| 구현 | 데이터 기반 효과로 구현 가능한가? |

## 직업 평가 기준

| 항목 | 기준 |
| --- | --- |
| 역할 명확성 | 한 문장으로 전투 역할을 설명할 수 있어야 한다 |
| 개별 덱 의미 | 해당 직업의 손패를 선택할 이유가 있어야 한다 |
| 공용 코스트 경쟁 | 다른 직업 카드와 코스트 경쟁을 만들어야 한다 |
| 필살기 조건 | 직업의 기본 플레이와 자연스럽게 연결되어야 한다 |
| 카드 5장 구성 | 공격/방어/유틸/시너지 중 최소 3축이 있어야 한다 |

## 합동기 평가 기준

합동기는 카드 입력 순서 패턴을 맞췄을 때 발동하는 소규모 시너지 보너스다.

좋은 합동기:

- 두 명 이상의 파티원이 의미 있게 참여한다.
- 공용 코스트 배분 고민을 만든다.
- 행동 순서와 입력 패턴이 중요하다.
- 발동 조건을 UI로 설명할 수 있다.
- 발동하지 않아도 기본 전투가 성립한다.
- 보너스가 작아 일반 카드와 필살기의 역할을 침범하지 않는다.

나쁜 합동기:

- 조건이 너무 복잡해 우연히만 발동된다.
- 특정 카드 조합만 강제한다.
- 발동하지 않으면 전투가 불가능하다.
- 일반 카드 선택을 무의미하게 만든다.

## 필살기 평가 기준

필살기는 일반 카드의 강화판이며, 자동 승격으로 발동한다.

평가 기준:

| 항목 | 기준 |
| --- | --- |
| 조건 명확성 | 플레이어가 조건을 의식하고 준비할 수 있어야 한다 |
| 자동 승격 적합성 | 이벤트 실행 시점에 자연스럽게 승격되어야 한다 |
| 스택 관리 | 스테이지 내부 유지, 발동 후 소비 규칙이 명확해야 한다 |
| 연출 차별 | 일반 카드와 구분되는 최소 연출이 있어야 한다 |
| 밸런스 | 반복 발동으로 일반 카드를 압도하면 안 된다 |

마감이 앞당겨질 경우 필살기는 추후 구현으로 이관할 수 있다.

## 몬스터 평가 기준

몬스터는 플레이어에게 특정 판단을 요구해야 한다.

| 타입 | 요구 판단 |
| --- | --- |
| 기본 공격형 | 방어와 공격의 기본 균형 |
| 방어/버프형 | 우선 처치 또는 무시 판단 |
| 디버프형 | 힐러/정화 카드의 가치 |
| 광역 공격형 | 파티원별 방어도 관리 |
| 엘리트 | 덱/파티 빌드 검증 |
| 보스 | 스테이지에서 쌓은 전략 검증 |

## 보스 기믹 설계 원칙

보스 기믹은 플레이어의 행동을 특정 방향으로 유도해야 하지만, 단 하나의 행동만 강제하면 안 된다.

좋은 보스 기믹:

- 플레이어에게 위험을 명확히 보여준다.
- 여러 대응 선택지를 남긴다.
- 방어, 공격, 공격+방어, 디버프, 회복 같은 다양한 해결책이 가능하다.
- 특정 행동을 유도하되, 그 행동만 정답으로 만들지 않는다.
- 코스트 감소, 손패 제한, 대상 제한처럼 선택 폭을 줄이는 압박은 가능하다.

나쁜 보스 기믹:

- “이번 턴은 방어만 해야 함”처럼 단일 행동을 강제한다.
- 특정 카드가 없으면 거의 무조건 패배한다.
- 기믹 설명이 부족해 맞고 나서야 이해된다.
- 플레이어의 덱빌딩 선택을 무효화한다.

## 보스 기믹 예시 기준

### 큰 피해 예고

좋은 선택지:

- 방어 카드 중심으로 버틴다.
- 공격으로 보스를 처치하거나 패턴을 끊는다.
- 공격 + 방어를 섞어 피해를 줄인다.
- 디버프로 보스 피해를 낮춘다.
- 힐러 카드로 맞은 뒤 회복할 계획을 세운다.

나쁜 설계:

```text
이번 턴 방어 카드만 사용하지 않으면 즉사
```

허용 가능한 압박:

```text
이번 턴 전체 코스트 -1
이번 턴 특정 파티원의 카드 비용 증가
이번 턴 보스가 공격받으면 다음 턴 피해 증가
```

이런 압박은 선택 폭을 줄이지만, 플레이어가 여러 방식으로 대응할 여지를 남긴다.
