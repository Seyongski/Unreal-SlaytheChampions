# level-rules.md

## 목적

로그라이크 덱빌딩 런의 난이도 곡선, 적 배치, 보상, 파티 성장 규칙을 정의한다.

## 런 구조 초안

```text
Act 1: 기본 덱과 파티 역할 학습
Act 2: 파티 시너지와 상태이상 대응 요구
Act 3: 완성된 빌드 검증
Final: 보스 또는 특수 엔딩 전투
```

## 스테이지 그리드 진행

한 스테이지는 여러 개의 그리드 노드로 구성된다. 플레이어는 그리드 안에서 다음 노드를 선택하며 전투, 이벤트, 상점, 휴식, 엘리트, 보스 등으로 이동한다.

스테이지 내부에서 유지되는 값과 스테이지 클리어 시 초기화되는 값을 구분한다.

| 범위 | 유지되는 요소 | 초기화 타이밍 |
| --- | --- | --- |
| 전투 내부 | 손패, 방어도, 일시 버프, 적 의도 | 전투 종료 |
| 스테이지 내부 | 필살기 조건 스택, 협동 기믹 카운트, 스테이지 한정 효과 | 스테이지 클리어 |
| 런 전체 | 덱, 유물, 장비, 파티 구성, 영구 강화 | 런 종료 |

필살기 조건 스택은 스테이지 내부에서는 유지된다. 플레이어는 여러 노드에 걸쳐 조건을 쌓고, 중요한 전투에서 자동 승격 필살기를 터뜨릴 수 있다.

## 노드 타입

| 노드 | 설명 |
| --- | --- |
| Battle | 일반 전투 |
| Elite | 고위험 고보상 전투 |
| Rest | 회복 또는 카드 강화 |
| Event | 선택형 이벤트 |
| Shop | 카드, 유물, 제거, 회복 구매 |
| Recruit | 파티원 영입 또는 교체 |
| Boss | 액트 보스 |

## 파티 규모 규칙

- 최소 파티원 수는 1명이다.
- 최대 파티원 수는 초기 설계상 3명을 기준으로 검토한다.
- 파티원이 많아질수록 카드 선택지는 늘지만 적의 광역 공격과 상태이상 압박도 증가한다.
- 파티원 수가 늘어날 때 UI 복잡도가 급격히 증가하므로 MVP에서는 최대 3명을 권장한다.

## 적 배치 규칙

| 전투 단계 | 적 수 | 설계 의도 |
| --- | --- | --- |
| 초반 일반전 | 1~2마리 | 카드 이벤트 큐와 기본 방어 학습 |
| 중반 일반전 | 2~3마리 | 대상 선택과 광역기 가치 학습 |
| 엘리트 | 1마리 강적 또는 2마리 조합 | 특정 대응 능력 검증 |
| 보스 | 1마리 중심 + 소환/패턴 | 빌드 완성도 검증 |

## 난이도 상승 축

난이도는 단순 체력 증가보다 행동 요구 조건을 늘리는 방식으로 올린다.

- 적 수 증가
- 방어 요구량 증가
- 상태이상 대응 요구
- 특정 턴까지 처리해야 하는 위협 추가
- 파티원별 피해 분산 압박
- 광역 공격 또는 집중 공격 패턴 혼합

## 보상 규칙

전투 종료 후 보상 후보:

- 카드 3장 중 1장 선택. 전투 후 필수 1회
- 골드. 전투 후 필수 지급
- 유물 또는 장비
- 카드 강화 재료
- 파티원 회복
- 포션 같은 사용 아이템. 랜덤 확률로 추가 지급
- 희귀한 경우 파티원 영입 기회

기본 덱빌딩 원칙:

- 보상 카드는 현재 파티 구성과 시너지가 있어야 한다.
- 지나치게 많은 카드를 강제로 받게 만들지 않는다.
- 카드 제거와 강화 수단을 충분히 제공한다.
- 한 런 또는 한 스테이지 안에서 최소 1장은 제거할 수 있어야 한다.

### 노드별 확정 보상

| 노드 | 확정 보상 | 추가 보상 |
| --- | --- | --- |
| Battle | 골드, 카드 보상 1회 | 낮은 확률로 사용 아이템 |
| Elite | 골드, 카드 보상 1회, 유물 1개 | Rare 카드 확률 증가 |
| Boss | 골드, 전설 카드 1장 | 스테이지 클리어 보상 |

보상 후보는 골드, 카드, 유물 중 랜덤한 N가지로 구성할 수 있다. 단, 전투 후 골드와 카드 보상 1회는 기본 보상으로 유지한다.

### 카드 등급 보상 규칙

카드 보상은 `Normal`, `Rare`, `Legendary` 등급을 사용한다.

| 등급 | 보상 내 역할 | 기본 노출 기준 |
| --- | --- | --- |
| Normal | 기본 덱 보강 | 일반 전투에서 가장 자주 등장 |
| Rare | 빌드 방향 강화 | 엘리트, 이벤트, 중후반 보상에서 비중 증가 |
| Legendary | 런의 핵심 변곡점 | 보스, 특수 이벤트, 매우 낮은 확률 보상 |

MVP에서는 카드 수가 적으므로 확률보다 `노드 타입별 보상 풀`을 우선한다.

권장안:

- 일반 전투: Normal 중심, 낮은 확률로 Rare
- 엘리트 전투: Rare 보장 또는 Rare 확률 증가, 유물 확정
- 보스 전투: Legendary Card 확정
- 특수 이벤트: 조건부 Legendary 또는 Special Card 제공

## 파티 성장 규칙

파티 성장은 카드 성장과 분리되지만 서로 영향을 줘야 한다.

가능한 성장 축:

- 파티원 영입
- 파티원 고유 카드 해금
- 파티원 패시브 강화
- 역할 특화 선택
- 전투 중 행동 주체 변경 카드 획득

## MVP 권장 범위

- 파티원 2명
- 적 최대 3마리
- Act 1 분량
- 일반전 6종, 엘리트 2종, 보스 1종
- 카드 25장 내외
- 상태이상 4~6종
- 유물 10~15종

## 미정 항목

- 파티원 영입이 런 중 랜덤 보상인지, 고정 이벤트인지
- 사망한 파티원의 처리 방식
- 액트 수와 최종 보스 구조
- 카드 보상 풀이 파티원별로 분리되는지
