# genre-tropes.md

## 목적

이 문서는 로그라이크 덱빌딩 장르에서 유저가 기대하는 기본 문법과, 이 프로젝트가 해당 문법을 어떻게 변형하는지 정리한다.

## 장르 핵심 문법

로그라이크 덱빌딩 게임은 아래 요소를 기본 기대값으로 가진다.

- 전투 후 카드 보상
- 덱 압축 또는 카드 제거
- 골드 보상
- 유물 또는 장비 보상
- 엘리트 전투의 고위험 고보상
- 보스 전투의 큰 보상
- 상점
- 휴식/회복
- 이벤트 선택지
- 위험과 보상의 갈림길

## 이 프로젝트의 보상 기본 규칙

### 전투 후 기본 보상

전투 후에는 보상이 반드시 발생한다.

기본 규칙:

- 전투 후 카드 보상은 필수 1회 제공한다.
- 전투 후 골드는 무조건 지급한다.
- 포션 같은 사용 아이템은 랜덤 확률로 추가 지급한다.
- 추가 보상은 골드, 카드, 유물 중 랜덤한 N가지 형태로 제공할 수 있다.

보상 예시:

```text
일반 전투 종료
-> 골드 지급
-> 카드 보상 3장 중 1장 선택
-> 낮은 확률로 포션 지급
```

### 카드 보상

- 전투 후 카드 보상은 필수 1회다.
- 기본 형태는 카드 3장 중 1장 선택이다.
- 카드 보상은 현재 파티 구성과 직업 덱을 고려해야 한다.
- 카드 등급은 Normal, Rare, Legendary를 사용한다.

### 덱 압축

덱 압축은 반드시 제공한다.

기본 규칙:

- 한 런 또는 한 스테이지 안에서 최소 1장은 제거할 수 있어야 한다.
- 카드 제거는 상점, 이벤트, 휴식 노드 중 하나 이상에서 제공한다.
- 덱 압축이 없으면 파티원별 개별 덱이 너무 빨리 비대해질 수 있다.

### 엘리트 보상

엘리트 전투는 확정 유물 보상을 제공한다.

기본 규칙:

- 엘리트 승리 시 유물 1개 확정 지급
- 골드는 일반 전투보다 높게 지급 가능
- 카드 보상은 일반 전투보다 Rare 확률을 높인다

### 보스 보상

보스 전투는 확정 전설 카드 보상을 제공한다.

기본 규칙:

- 보스 승리 시 Legendary Card 1장 확정 드랍
- 보스 보상은 스테이지 클리어 보상과 연결된다
- 보스 보상은 다음 스테이지의 빌드 방향을 바꿀 수 있어야 한다

## 사용 아이템

포션 같은 사용 아이템은 전투 보상에서 랜덤 확률로 추가된다.

역할:

- 짧은 세션에서 즉시 판단을 만든다.
- 특정 전투의 위기를 넘기는 안전장치가 된다.
- 덱에 직접 들어가지 않아 덱 비대화를 막는다.

MVP에서는 포션 시스템을 깊게 만들지 않는다. 단순 회복, 방어도 증가, 카드 드로우, 코스트 회복 정도를 우선한다.

## 이 프로젝트의 장르 변형

| 장르 문법 | 일반적인 형태 | 이 프로젝트의 변형 |
| --- | --- | --- |
| 덱 | 단일 덱 | 파티원별 개별 덱 |
| 코스트 | 플레이어 1인 자원 | 파티 공용 코스트 |
| 카드 사용 | 즉시 효과 중심 | Action Event 큐 등록 |
| 고점 보상 | 강한 카드/유물 | 조건 충족 필살기 자동 승격 |
| 진행 | 맵 노드 진행 | 타워 내부 그리드 스테이지 |
| 누적 조건 | 전투 내부 중심 | 스테이지 내부 필살기 조건 스택 유지 |

## 지켜야 할 기대감

유저가 로그라이크 덱빌딩에서 기대하는 감각은 유지해야 한다.

- 전투 후 항상 뭔가를 얻는다.
- 덱이 런 중 변한다.
- 좋은 보상을 위해 위험한 전투를 선택할 수 있다.
- 보스는 빌드 검증 지점이어야 한다.
- 상점/이벤트/휴식이 전투 사이의 리듬을 만든다.
- 카드 제거가 덱 품질 관리 수단으로 존재한다.

## 피해야 할 장르 함정

- 카드 보상이 너무 많아 덱이 빠르게 망가지는 구조
- 덱 압축 수단이 부족해 원하지 않는 카드가 계속 쌓이는 구조
- 엘리트 보상이 약해 위험을 감수할 이유가 없는 구조
- 보스 보상이 약해 스테이지 클리어 감각이 부족한 구조
- 전투 후 보상이 매번 비슷해지는 구조
- 파티원별 덱 때문에 보상 선택지가 너무 복잡해지는 구조
- 필살기만 기다리면 되는 구조
