# world-setting.md

## 목적

이 문서는 게임의 세계관, 톤앤매너, 직업/카드/몬스터 명칭을 정할 때의 기준을 정의한다.

카드와 스킬의 구체적인 이름은 추후 별도 작업에서 추가한다. 현재 문서는 최초 마스터의 방향을 잡기 위한 톤 가이드로 사용한다.

## 핵심 콘셉트

중세풍 판타지 세계관을 기반으로 한 타워 등반 이야기.

플레이어의 파티는 같은 왕국이나 길드 출신이 아니라, 각자 다른 차원에서 불려온 인물들로 구성된다. 이들은 서로 다른 세계의 전투 방식, 마법, 신앙, 기술을 가지고 탑을 오른다.

```text
다른 차원의 인물들
-> 탑에 소환 또는 유입
-> 서로의 목적은 다르지만 생존과 탈출을 위해 협력
-> 층을 오르며 카드와 힘을 축적
-> 탑의 정상 또는 중심부에 가까워짐
```

## 톤앤매너

### 기본 톤

- 중세풍 판타지
- 어둡지만 지나치게 암울하지 않음
- 모험과 생존의 긴장감
- 다양한 차원의 이질감
- 탑을 오를수록 세계가 뒤틀리는 분위기

### 피해야 할 톤

- 현대 코미디풍 대사
- 너무 밝은 캐주얼 판타지
- 지나치게 복잡한 고유명사 남발
- SF 기계문명 중심 설정
- 카드 효과를 이해하기 어렵게 만드는 과한 설정어

## 탑 설정

탑은 여러 차원과 연결된 거대한 구조물이다.

탑의 특징:

- 층마다 다른 공간과 규칙을 가진다.
- 다른 차원의 존재들이 탑 안으로 끌려온다.
- 탑은 도전자에게 카드, 유물, 저주, 시험을 제공한다.
- 스테이지 그리드는 탑 내부의 갈림길과 방을 표현한다.
- 스테이지 클리어는 한 층 또는 한 구역을 돌파하는 의미를 가진다.

## 파티원 설정

파티원은 각자 다른 차원에서 왔다.

이 설정은 아래 시스템을 자연스럽게 설명한다.

| 시스템 | 세계관 해석 |
| --- | --- |
| 파티원별 개별 덱 | 각자의 차원과 전투 방식이 다르기 때문 |
| 직업 전용 카드 | 각 인물의 세계에서 익힌 고유 기술 |
| 공용 카드 | 탑 안에서 배운 기본 전투법 또는 공통 생존술 |
| 특수 카드 | 탑의 장치, 보스 기믹, 특정 방의 규칙 |
| 저주 카드 | 탑이 부여하는 오염, 맹세, 기억의 파편 |
| 필살기 | 각 차원의 힘이 특정 조건에서 폭발하는 순간 |

## 최초 직업 3종

### 검사

다른 차원에서 온 근접 전투 전문가.

역할:

- 단일 대상 공격
- 강화 누적
- 강한 한 방
- 필살기 자동 승격의 대표 사례

톤:

- 절제된 검술
- 전사의 집중
- 한 번의 참격으로 판도를 바꾸는 이미지

### 마법사

다른 차원의 마법 체계를 다루는 캐스터.

역할:

- 광역 피해
- 필드 효과
- 디버프
- 원소 또는 공간 변화

톤:

- 오래된 주문
- 차원의 균열
- 필드 전체를 바꾸는 마법

### 힐러

다른 차원의 신앙, 생명술, 회복 마법을 사용하는 지원가.

역할:

- 회복
- 보호
- 정화
- 파티 버프

톤:

- 신성함
- 희생과 보호
- 무너진 전투를 다시 세우는 힘

## 필살기 세계관 해석

필살기는 단순한 궁극기 버튼이 아니다.

각 파티원이 자신의 차원에서 가져온 힘이, 특정 조건을 만족했을 때 탑의 규칙과 공명하며 폭발하는 현상이다.

예시:

```text
검사가 강화 상태를 여러 번 쌓음
-> 탑의 전투 규칙이 검사의 힘을 증폭
-> 공격 이벤트가 필살기로 자동 승격
```

이 설정은 자동 승격 시스템과 잘 맞는다. 플레이어가 버튼을 누르는 것이 아니라, 조건이 맞는 순간 탑과 캐릭터의 힘이 반응해 발동하는 느낌을 준다.

## 몬스터와 보스 톤

몬스터는 탑의 시험, 다른 차원의 잔재, 탑에 갇힌 존재들로 구성한다.

몬스터 방향:

- 갑옷을 입은 망령
- 탑의 수호자
- 저주받은 짐승
- 차원 균열에서 나온 생물
- 마법에 오염된 병사
- 탑의 장치와 결합한 보스

보스는 플레이어를 단순히 강한 수치로 압박하기보다, 특정 행동을 유도하는 시험처럼 설계한다.

## 네이밍 가이드

카드와 스킬 이름은 추후 추가한다.

현재 기준:

- 직관적인 효과를 해치지 않는다.
- 중세풍 판타지 어휘를 기본으로 한다.
- 차원, 탑, 맹세, 균열, 축복, 저주, 검, 룬, 성흔 같은 단어를 활용할 수 있다.
- 카드명만 보고 대략적인 효과 분위기가 떠올라야 한다.
- 너무 긴 고유명사는 피한다.

## 최초 마스터에서 필요한 세계관 표현

마스터 빌드에서는 긴 스토리보다 아래 요소를 우선한다.

- 메인 메뉴 또는 시작 문구에서 탑 등반 목표 제시
- 검사/마법사/힐러가 다른 차원 출신임을 짧게 암시
- 스테이지 그리드가 탑 내부 진행처럼 느껴지게 표현
- 보스가 탑의 시험이라는 인상을 주기
- 카드/필살기 UI에서 중세 판타지 톤 유지

## 미정 항목

- 탑의 이름
- 각 차원의 이름
- 검사/마법사/힐러의 고유 캐릭터명
- 탑에 오르는 최종 목적
- 최종 보스의 정체
- 카드와 스킬의 구체 이름
