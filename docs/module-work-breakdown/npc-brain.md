# NPC Brain

## 역할

적의 행동 규칙, 의도 표시, 몬스터 패턴, 엘리트/보스 기믹을 담당한다.

## 담당 범위

- Intent System
- Pattern AI
- 일반 몬스터, 엘리트, 보스 패턴
- 보스 특수 기믹
- 플레이어 선택을 유도하되 강제하지 않는 패턴 설계

## 하위 시스템

| 하위 시스템 | 내용 | 완료 기준 |
| --- | --- | --- |
| Intent System | 다음 턴 적 행동을 플레이어에게 미리 보여준다 | 적의 다음 행동이 HUD에 표시된다 |
| Pattern AI | 단순 공격, 방어, 디버프, 조건부 패턴 변화 | 몬스터마다 행동 리듬이 다르다 |
| Boss Gimmick | 특정 조건에서 패턴이 변화하되 정답 행동을 강제하지 않는다 | 방어/공격/복합 대응이 모두 선택지로 남는다 |

## Phase별 목표

| Phase | 목표 |
| --- | --- |
| Phase 1 | 적 1~2종, 단순 공격 AI |
| Phase 2 | 몬스터 3종 이상, 공격/방어/디버프 패턴 |
| Phase 3 | 엘리트/보스 노드와 적 데이터 연결 |
| Phase 4 | 몬스터 5~7종, 엘리트 1~2종, 보스 1종 |
| Phase 5 | 패턴 난이도와 기믹 가독성 조정 |
